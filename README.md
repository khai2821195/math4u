# Math4U 데이터 저장소

본 저장소는 카이 Math4U 학원의 수학 기출문제(PDF) 및 출제 분석(HTML) 자료를 체계적으로 관리하기 위한 공간입니다.

## 저장소 구조
- `/2026/1-1/H2`: 2026학년도 1학기 1차 고사 2학년 관련 자료
  - `.html`: 시험 출제 경향 및 상세 분석 보고서
  - `.pdf`: 기출문제 자료 ('반전' 처리된 선명한 버전)

## 사용 안내
각 파일은 블로그 및 학원 내부 시스템에서 링크로 연결하여 조회할 수 있도록 최적화되어 있습니다. 데이터는 학기별/학교별로 지속적으로 업데이트됩니다.
